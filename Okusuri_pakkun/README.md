# ぱっくんスタンプ (Pakkun Stamp)

お子様の服薬を楽しく習慣化するためのWebアプリケーションです。

## 概要
ナッジ理論と児童心理学に基づき、服薬後のポジティブなフィードバック（演出と褒め言葉）を通じて、お子様が自発的に、かつ楽しくお薬を飲めるようサポートします。

## 主な機能
- **スタンプカード**: 服薬スケジュールに合わせたスタンプ管理。
- **サプライズ演出**: スタンプ時の多彩なアニメーションと効果音。
- **ペアレンタル・ナッジ**: 保護者向けの「褒め言葉」提案機能。
- **表彰状**: 完了時のデジタル表彰状とA4印刷機能。
- **マルチ管理**: 最大3つのスケジュールを同時管理。

## 開発者向け
ローカルサーバーを立てて実行するには、以下のコマンドを使用してください：

```bash
# Pythonを使用する場合
python -m http.server 5001

# Node.js (serve) を使用する場合
npx serve .
```

ブラウザで `http://localhost:5001` (または指定されたポート) を開いてください。
