注意事項

  • 本ツールは、厚生労働省が公開しているExcelファイルに基づいて情報を表示します。データの正確性については、厚生労働省のウェブサイトで最新の情報を確認してください。
  • 本ツールは情報収集のサポートを目的としており、その正確性や完全性を保証するものではありません。ご利用によって生じるいかなる結果についても、責任を負いかねますのでご了承ください。
  • 本ツールの著作権は開発者に帰属します。無断での複製、配布、販売は禁止します。
  • PMDAのリダイレクトページ上の医薬品ガイド、RMP資材、電子化された添付文書等の著作権は製造販売業者が有します。無断での複製、転載、頒布は禁じられます。

医薬品出荷状況検索ツール

  • 検索条件の入力

    「医薬品名」「成分名」「規格、剤形、メーカー」の各項目に、検索したいキーワードを入力します。複数の項目にキーワードを入力することで、AND検索が可能です。各検索欄に複数のキーワードをスペース（全角・半角）区切りで入力すると、全てのキーワードが含まれる医薬品を絞り込むことができます。

  • 出荷状況での絞り込み

    「出荷状況」の欄にあるチェックボックスで、「通常出荷」「出荷制限」「供給停止」の表示/非表示を切り替えることができます。複数のチェックボックスにチェックを入れることで、表示したい状況を組み合わせることが可能です。

  • 検索の実行

    検索欄にキーワードを入力するだけで、自動的に検索結果が表示されます。リアルタイムで結果が更新されるため、スムーズにご利用いただけます。

  • 検索結果からの便利な操作

    検索結果の品名をクリックすると、便利な操作メニューが開きます。

    • 医薬品情報: PMDA（医薬品医療機器総合機構）の医薬品情報ページにアクセスします。
    • 代替薬検索: 該当の医薬品のYJコードを基に、代替薬候補を検索します。
    • ヒヤリハット検索: 該当の医薬品に関連するヒヤリ・ハット事例を検索します。

    また、成分名をクリックすると、検索欄がクリアされ、その成分名での検索が実行されます。

  • ソート

    「出荷状況」の項目をクリックすると、検索結果を「出荷状況」で並べ替えることができます。

情報更新日検索ツール

  • 検索条件の入力

    「品名、成分名で検索」欄にキーワードを入力します。複数のキーワードをスペース（全角・半角）区切りで入力すると、全てのキーワードが含まれる医薬品を絞り込むことができます。

  • 出荷状況での絞り込み

    「出荷状況」の欄にあるチェックボックスで、「通常出荷」「限定出荷」「供給停止」の表示/非表示を切り替えることができます。複数のチェックボックスにチェックを入れることで、表示したい状況を組み合わせることが可能です。

  • 情報更新日での絞り込み

    「情報更新日」の欄にあるチェックボックスで、指定した期間内に情報が更新された医薬品のみを表示できます。「すべて」を選択すると期間による絞り込みを解除します。

  • 検索の実行

    検索欄にキーワードを入力するか、フィルター条件を変更すると、自動的に検索結果が表示されます。「検索」ボタンをクリックしても実行されます。

  • ソート

    「品名」または「更新日」の項目をクリックすると、検索結果を並べ替えることができます。

YJコード検索ツール

  • YJコードでの検索

    YJコード入力欄に12桁のYJコードを入力し、「検索」ボタンをクリックするかEnterキーを押すと、該当する医薬品情報が表示されます。

  • 検索項目での絞り込み

    「検索項目」のチェックボックスで、YJコードのどの桁数までを検索条件とするかを指定できます。例えば、「薬効群3桁」にチェックを入れると、入力されたYJコードの先頭3桁が一致する医薬品を検索します。

  • 出荷状況での絞り込み

    「出荷状況」のチェックボックスで、「通常出荷」「限定出荷」「供給停止」の表示/非表示を切り替えることができます。複数のチェックボックスにチェックを入れることで、表示したい状況を組み合わせることが可能です。

  • 検索結果からの便利な操作

    検索結果の品名をクリックすると、その医薬品のYJコードが検索欄に自動入力され、再度検索が実行されます。

薬局ヒヤリ・ハット事例ビューア

  • 柔軟な検索機能

    「医薬品名」と「絞り込み（フリーワード）」の2つの入力欄を使って、柔軟な検索が可能です。検索はサーバーサイドで実行され、常に新しい事例から順に最大100件が取得されます。

    • 医薬品名のみ入力: 医薬品名または成分名で検索します。
    • 絞り込みのみ入力: 事例内容全体からフリーワードで検索します。
    • 両方入力: 医薬品名とフリーワードの両方を含む事例を検索します（AND検索）。

  • 段階的な結果表示

    検索結果は、まず30件が表示されます。取得した結果が30件より多い場合、リストの下に「さらに表示」ボタンが出現します。このボタンをクリックすることで、次の30件を順次読み込むことができ、快適なブラウジングが可能です。

  • ランダム表示

    「ランダム」ボタンをクリックすると、現在の検索結果または全事例の中からランダムに事例が表示されます。

  • 事例の詳細表示

    各事例カードには、発生年月、事例の詳細、背景・要因、改善策などが分かりやすく表示されます。

このツールの目的と主な機能

  • 医薬品出荷状況検索ツール

    厚生労働省が公開している医薬品の出荷状況データを効率的に検索するためのツールです。医薬品名、成分名、規格、剤形、メーカー、出荷状況で絞り込み検索が可能です。

  • 情報更新日検索ツール

    医薬品の出荷状況データの中から、特定の期間内に情報が更新された医薬品を検索・表示するためのツールです。最新の情報を素早く確認したい場合に便利です。

  • YJコード検索ツール

    YJコードに基づいて医薬品情報を検索するためのツールです。YJコードの桁数による絞り込みや、出荷状況でのフィルターが可能です。

  • 薬局ヒヤリ・ハット事例ビューア

    薬局におけるヒヤリ・ハット事例を検索・閲覧するためのツールです。医薬品名やキーワードで事例を絞り込み、ランダム表示も可能です。

  • 共通機能

    自動データ更新: 各ツールは、厚生労働省が公開しているデータを自動的に更新します。ユーザーが手動でファイルをアップロードする必要はありません。

    文字の正規化: 全角・半角、ひらがな・カタカナ、大文字・小文字を自動的に統一します。

    レスポンシブデザイン: PCだけでなく、スマートフォンでも快適に利用できるように設計されています。
