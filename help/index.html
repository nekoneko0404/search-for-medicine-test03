Kusuri Compass 取扱説明書

医薬品の情報を、もっと速く、もっと分かりやすく。

機能の全体像

Kusuri Compassは、4つのツールが連携して医薬品情報を多角的に提供します。

🚀 クイックスタート

最も基本的な「医薬品の出荷状況」を調べる方法です。

1
キーワード入力: トップページの「医薬品名」または「成分名」に、探している薬の名前を入力します。入力の途中でも自動で検索が始まります。
2
結果の確認: 該当する医薬品が一覧で表示されます。「出荷状況」の列で、現在の供給ステータス（通常出荷、限定出荷、供給停止）を確認できます。
3
詳細情報へ: さらに詳しい情報が必要な場合は、品名をクリックすると表示されるメニューから「医薬品情報(PMDA)」や「代替薬検索」など、次のアクションに進めます。

医薬品出荷状況検索

できること

医薬品の品名や成分名から、現在の出荷状況を素早く検索できます。このツールがKusuri Compassの入り口です。

操作手順:

  1. 「医薬品名」「成分名」「規格、剤形、メーカー」のいずれか、または複数にキーワードを入力します。
  2. 入力と同時に、下部に検索結果が自動で表示されます。
  3. 「出荷状況」のチェックボックスで、表示するステータスを絞り込めます。
  4. 結果の品名をクリックすると、さらに詳細な情報（添付文書、代替薬、ヒヤリハット事例など）へアクセスできます。

情報更新日検索

できること

「最近、供給状況が変わった薬は？」を知りたい時に使います。情報の更新日を基準に検索できます。

操作手順:

  1. トップページ右上の「情報更新日検索」ボタンから移動します。
  2. 「情報更新日」フィルターで、「3日以内」や「7日以内」など、期間を選択します。
  3. 必要に応じて、品名や出荷状況でさらに絞り込みます。
  4. 最近情報が更新された医薬品が一覧で表示されます。特に供給が再開された薬などを探すのに便利です。

YJコード検索

できること

供給停止中の薬の代替薬を探すのに役立ちます。YJコードを使い、薬効や剤形が似ている薬を検索します。

操作手順:

  1. 出荷状況検索の結果などから、このページに移動します。YJコードは自動で入力されています。
  2. 「検索項目」で、どの範囲で類似薬を探すかを選択します。（例：「薬効群4桁」や「成分」が一致する薬）
  3. 「検索」ボタンを押すと、代替薬の候補が一覧で表示されます。
  4. 結果から「通常出荷」されている薬を選ぶことで、代替薬を見つけやすくなります。

薬局ヒヤリ・ハット事例ビューア

できること

医薬品に関連するヒヤリ・ハット事例（インシデントレポート）を検索・閲覧できます。

操作手順:

  1. 出荷状況検索の結果などから、このページに移動します。医薬品名は自動で入力されています。
  2. 「絞り込み」にキーワード（例：「粉砕」「小児」など）を追加して、より詳細な事例を検索できます。
  3. 検索結果はカード形式で表示されます。「さらに表示」で続きを読むことができます。

データソース: このツールは、公益財団法人日本医療機能評価機構が提供する「薬局ヒヤリ・ハット事例収集・分析事業」の公式APIを使用しています。データはプロキシ (`https://hiyari-proxy-708146219355.asia-east1.run.app/proxy`) を経由して取得されます。

便利なTips

複数キーワードで絞り込み: 検索欄にスペース区切りで複数の単語（例：「ロキソプロフェン 60mg」）を入れると、両方を含むものだけを検索できます（AND検索）。

アイコンの意味:
後: 後発医薬品 (ジェネリック)
基: 基礎的医薬品
▲ / ▼: 供給状況の変動を表します。（情報更新日検索ツール）

データが古い？: 画面右下の 🔄 ボタンを押すと、キャッシュがクリアされ、最新のデータが強制的に読み込まれます。
