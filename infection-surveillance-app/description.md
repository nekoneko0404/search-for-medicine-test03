### 地図の使い方
地域をクリックすると、その地域の都道府県ごとの数値が右側のパネルに表示されます。
都道府県名をクリックすると、その都道府県の過去の推移グラフが表示されます。
（※ARIはグラフ表示に対応していません）

### 色の基準
各疾患の定点当たり報告数に基づいて、以下の基準で色分けを行っています。

**インフルエンザ**
- <span style="color:#e74c3c">■</span> 30.0以上: 警報レベル
- <span style="color:#f39c12">■</span> 10.0以上: 注意報レベル
- <span style="color:#f1c40f">■</span> 1.0以上: 流行入り
- <span style="color:#2ecc71">■</span> 1.0未満: 平常

**COVID-19**
- <span style="color:#e74c3c">■</span> 15.0以上: 警報レベル相当
- <span style="color:#f39c12">■</span> 10.0以上: 注意報レベル相当
- <span style="color:#2ecc71">■</span> 10.0未満: 平常

**急性呼吸器感染症 (ARI)**
- <span style="color:#e74c3c">■</span> 120.0以上: 流行レベル
- <span style="color:#f39c12">■</span> 80.0以上: 注意が必要
- <span style="color:#2ecc71">■</span> 80.0未満: 平常
