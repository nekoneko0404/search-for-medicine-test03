# 供給状況検索アプリケーション

## データソース

### カテゴリ分類
**情報ソース:** 供給確保医薬品の候補成分一覧（学会推薦内容）R7年度 カテゴリ分類案  
**URL:** https://www.mhlw.go.jp/content/10808000/001586778.pdf

カテゴリデータ (`data/category_data.json`) は上記PDFから抽出した成分名とカテゴリ分類（A, B, C）を含みます。

- **カテゴリA**: 最も優先して取り組む安定確保医薬品（21成分）
- **カテゴリB**: 優先して取り組む安定確保医薬品（29成分）
- **カテゴリC**: 安定確保医薬品（456成分）

### 供給状況データ
供給状況データは、既存のGoogleスプレッドシートから取得しています（`../js/data.js`経由）。

## ファイル構成

```
supply-status/
├── index.html          # メインHTMLファイル
├── script.js           # JavaScriptロジック
├── data/
│   └── category_data.json  # カテゴリ分類データ
└── README.md           # このファイル
```

## 使用方法

1. ローカルサーバーを起動:
   ```bash
   python -m http.server 8000
   ```

2. ブラウザで `http://localhost:8000/supply-status/index.html` にアクセス

3. 成分名で検索、カテゴリや出荷状況でフィルタリングが可能

4. 成分名をクリックすると、その成分に含まれる医薬品の詳細が表示されます
