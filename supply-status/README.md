# 供給状況検索アプリケーション

## データソース

### カテゴリ分類
**情報ソース:** 供給確保医薬品の候補成分一覧（学会推薦内容）R7年度 カテゴリ分類案  
**URL:** https://www.mhlw.go.jp/content/10808000/001586778.pdf

カテゴリデータ (`data/category_data.json`) は上記PDFから抽出した成分名とカテゴリ分類（A, B, C）を含みます。

- **カテゴリA**: 最も優先して取り組む安定確保医薬品（35エントリ）
- **カテゴリB**: 優先して取り組む安定確保医薬品（40エントリ）
- **カテゴリC**: 安定確保医薬品（579エントリ）

※同一成分が複数の薬効分類に分類される場合があるため、エントリ数（重複含む）で記載しています。ユニークな成分数は614成分です。

### 供給状況データ
供給状況データは、既存のGoogleスプレッドシートから取得しています（`../js/data.js`経由）。

supply-status/

## 機能

### サマリービュー
- カテゴリ別、薬効分類別に成分を一覧表示
- 各成分の通常出荷・限定出荷・供給停止の件数を表示
- カテゴリ（A/B/C）でフィルタリング可能
- 医薬品名・成分名で検索可能

### 詳細ビュー
- 成分名をクリックすると、その成分に含まれる医薬品の詳細を表示
- 出荷状況（通常出荷/限定出荷/供給停止）でフィルタリング可能
- 制限理由、解消見込み、見込み時期、更新日を確認可能

## 使用方法

1. ローカルサーバーを起動:
   ```bash
   python -m http.server 8000
   ```

2. ブラウザで `http://localhost:8000/supply-status/index.html` にアクセス

3. 成分名で検索、カテゴリや出荷状況でフィルタリングが可能

4. 成分名をクリックすると、その成分に含まれる医薬品の詳細が表示されます
