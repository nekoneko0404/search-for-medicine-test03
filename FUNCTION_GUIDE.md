# Kusuri Compass 機能紹介ガイド

Kusuri Compass（くすりコンパス）のメニューページから利用できる各種機能について紹介します。

## 1. 医薬品出荷状況検索
医薬品の供給状況をリアルタイムで検索・確認できる機能です。
- **主な特徴:**
    - 複数のメーカーの出荷状況（通常出荷、限定出荷、供給停止など）を一括検索。
    - Google Sheetsをデータソースとし、常に最新の情報を反映。
    - 同一成分の代替薬検索や、メーカー公式サイトへのダイレクトリンク機能を搭載。
    - ヒヤリハット検索とのスムーズな連携。

## 2. 出荷状況更新
医薬品の出荷状況に関する最新の更新情報を一覧で確認できる機能です。
- **主な特徴:**
    - 最近更新された品目を素早く把握。
    - 供給状況の変動（改善や悪化）を視覚的に表示。

## 3. 代替薬ナビゲーター
生成AI（Gemini）を活用して、欠品時の代替薬を提案する機能です。
- **主な特徴:**
    - 医薬品名を入力するだけで、AIが適切な代替候補を提示。
    - 供給不安定な状況下での処方提案や疑義照会をサポート。
    - ※生成AIの特性を理解し、最終的な判断は専門家が行う必要があります。

## 4. ヒヤリハット検索
公益財団法人日本医療機能評価機構が収集・公開している「薬局ヒヤリ・ハット事例」を検索できる機能です。
- **主な特徴:**
    - 医薬品名やキーワードで過去の事例を素早く調査。
    - 事例の概要、要因、改善策を確認し、リスクマネジメントに活用。
    - 医薬品検索結果から特定の薬剤に関する事例へ直接ジャンプ可能。

## 5. 全国感染状況
全国の感染症発生動向を地図とグラフで可視化する機能です。
- **主な特徴:**
    - 国立感染症研究所（IDWR）のデータを元に、インフルエンザやCOVID-19などの流行状況を表示。
    - 日本地図（ブロックマップ）による直感的な地域別流行把握。
    - 都道府県別の詳細データや、過去数年分との比較グラフを表示。

## 6. おくすりぱっくん
小児用医薬品の「飲み合わせ（味の相性）」を検索できる機能です。
- **主な特徴:**
    - 苦い薬を何に混ぜれば飲みやすくなるか、逆に混ぜてはいけないものは何かを調査。
    - チョコレート、アイスクリーム、ジュースなどとの相性を判定。
    - お子様の服薬指導を強力にサポート。

## 7. お薬飲めたね！
お子様が楽しく服薬を続けられるようにサポートする、スタンプカード形式のアプリです。
- **主な特徴:**
    - 薬を飲んだらキャラクターのスタンプをゲット。
    - 全部のスタンプが貯まると「ごほうび」の表彰状を表示。
    - 保護者向けの服薬管理・モチベーション維持ツール。

## 8. 花粉飛散状況（近日公開予定）
全国の花粉飛散状況をリアルタイムで監視・可視化する機能です。
- **主な特徴:**
    - ウェザーニューズ社の「ポールンロボ」観測データを活用。
    - 地図上でのリアルタイムな飛散量表示。
    - 気象データ（気温・風速など）と組み合わせたトレンド分析。

---
これらの機能は、薬剤師の日常業務の効率化と、患者様へのより良い情報提供を支援するために開発されています。

## データソースについて
本システムでは、信頼性の高い以下の公的機関や企業のデータを利用しています。

- **医薬品出荷状況:** 厚生労働省「[医薬品の供給状況等に関する資料](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/iryou/kouhatu-iyaku/04_00003.html)」のデータを加工して利用
- **ヒヤリハット事例:** 公益財団法人 日本医療機能評価機構「[薬局ヒヤリ・ハット事例収集・分析事業](https://www.med-safe.jp/hiyari/pharmacy/)」
- **感染症情報:** 国立感染症研究所「[感染症発生動向調査（IDWR）](https://www.niid.go.jp/niid/ja/idwr.html)」
- **花粉情報:** 株式会社ウェザーニューズ「[ポールンロボ](https://weathernews.jp/s/pollen/)」観測データ
- **気象データ:** [Open-Meteo API](https://open-meteo.com/)
- **代替薬提案:** [Google Gemini AI](https://gemini.google.com/) (Large Language Model)
- **飲み合わせデータ:** 各製薬メーカー添付文書・インタビューフォーム（[PMDA 医薬品情報検索](https://www.pmda.go.jp/PmdaSearch/iyakuSearch/)等）

## 技術スタック
モダンなWeb技術を組み合わせ、高速でメンテナンス性の高いシステムを構築しています。

### フロントエンド
- **HTML5 / JavaScript (ES6+):** 標準的なWeb技術による構築。
- **Tailwind CSS:** 迅速かつ柔軟なUIデザインの実現。
- **Chart.js:** 感染症推移や花粉飛散量の動的なグラフ描画。
- **localForage:** ブラウザ内データベース（IndexedDB）を活用した高速なキャッシュ機構。

### バックエンド / インフラ
- **Google Apps Script (GAS):** データの取得・加工およびAPIプロキシとして活用。
- **Google Sheets:** 柔軟なデータ管理用データベースとして利用。
- **Cloudflare Pages:** 高速でセキュアなホスティング環境。
- **Gemini API:** 高度な自然言語処理による代替薬の提案ロジック。
