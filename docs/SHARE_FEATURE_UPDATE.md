# 情報更新日検索 - 共有機能実装

## 実装内容

### 1. UI追加
- 検索条件エリアの「供給回復」ボタンの横に「共有」ボタンを追加
- 共有ボタンは indigo カラーのアウトラインスタイル
- 共有アイコン(シェアアイコン)を表示

### 2. 機能実装

#### クエリパラメータ仕様
以下のパラメータをURLに含めます:

| パラメータ | 説明 | 値 |
|----------|------|-----|
| `q` | 検索キーワード | 入力されたキーワード |
| `period` | 更新期間 | 3days, 7days, 14days, 30days, all |
| `normal` | 通常出荷フィルタ | 1=ON, 0=OFF |
| `limited` | 限定出荷フィルタ | 1=ON, 0=OFF |
| `stopped` | 供給停止フィルタ | 1=ON, 0=OFF |
| `up` | 傾向(改善)フィルタ | 1=ON, 0=OFF |
| `down` | 傾向(悪化)フィルタ | 1=ON, 0=OFF |

#### 共有機能
1. 共有ボタンをクリック
2. 現在の検索条件(キーワード + 期間 + 各フィルタ)をクエリパラメータとして生成
3. 完全なURLをクリップボードにコピー
4. 成功メッセージを表示

#### URL復元機能
1. ページ読み込み時にURLクエリパラメータをチェック
2. パラメータが存在する場合、検索条件を復元
3. 自動的に検索を実行

## 使用例

### 共有URLの例
```
update/index.html?q=アムロジピン&period=30days&normal=1&limited=1&stopped=0&up=1&down=0
```

この例では:
- キーワード: `アムロジピン`
- 期間: `30日以内`
- 通常出荷: ON
- 限定出荷: ON
- 供給停止: OFF
- 傾向(改善): ON
- 傾向(悪化): OFF

## 実装ファイル

- `update/index.html`: 共有ボタンのUI追加
- `update/js/main.js`: 
  - `shareSearchConditions()`: 共有機能
  - `restoreFromUrlParams()`: URL復元機能
  - `initApp()`: URL復元処理の統合
